# Frontend (Vite + React + TypeScript)

Quickstart

1. Install dependencies:

```bash
cd frontend
npm install
```

2. Start dev server:

```bash
npm run dev
```

3. Open http://localhost:5173/

Notes
- Project uses Vite and React Router. Main entry is `src/main.tsx`.
- Environment variables should be placed in `.env` (e.g. `VITE_API_URL`).
